ABOUT

私たちは、年齢や経験に関係なく勉強をして、一人前の技術者になる人を積極的に応援します。
そして、その中で生まれた人間関係を大切にする技術者コミュニティです。
実際に、私たちの中にはバーテンダーや美容師など、さまざまなバックグラウンドを持つ仲間がたくさんいます。
皆ゼロから学習を重ね、さまざまな現場でエンジニア・デザイナーとして働いています。
この度、まだ勉強を始めたばかりの方や現場にでたばかりの技術者同士のつながりを増やしたいと思い主催いたします！
